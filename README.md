# h-reader

[microformats2](http://microformats.org/wiki/microformats2) [feed](http://microformats.org/wiki/h-feed) reader.


## h-feed to Atom proxy

Get an Atom feed for any page that is marked up with [h-feed](http://microformats.org/wiki/h-feed).

```
http://h-reader.com/feed/atom?url=http://example.com
```
